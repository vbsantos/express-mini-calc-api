# Mini Calculadora

Projeto da etapa 1.

---

## Endpoints

### Documentation API at root endpoint

- `/`

### Operations endpoints

- `/api/v1/sum`
- `/api/v1/sub`
- `/api/v1/div`
- `/api/v1/mul`

### Validation endpoint

- `/api/v1/validate`

---

## Usage

To install dependencies

```bash
yarn
```

To run in dev mode

```bash
yarn dev
```

To run tests

```bash
yarn test
```

To build

```bash
yarn build
```

To run build

```bash
yarn serve
```
